# SoftLight
This is a homekit controlled light using an ESP8266 chip.
# Scope
The scope of this project is organizing my files. For now...it is not a community project - yet. Help and suggestions are welcome though!
# Why is this different?
I hate the way regular bulbs just turn on or off harshly. There are 1023 levels available through PWM, so why not make it a soft transition?
## Parts of this Project
There are multiple parts - I intent to just make a different folder for each until I can find a better way to combine them.
One of the parts is a library that includes various fundamental functions.
The other parts are actual implementations ready to flash on a board.
